# 📱 RED Mobile

React Native mobilna aplikacija za prikaz gužve u Zagrebu.

## O projektu

Mobilna verzija RED app — prikazuje trenutnu gužvu na 60+ lokacija u Zagrebu. Korisnici mogu prijaviti gužvu direktno s mobitela.

## Features

- 📍 Lista lokacija s trenutnom gužvom
- 🔴 Prijava gužve (visoka / umjerena / niska)
- 👥 Brojač prijava po lokaciji
- 🔄 Povezano s RED API-jem u realnom vremenu

## Tech stack

- React Native
- Expo
- TypeScript

## Kako pokrenuti

```bash
git clone https://github.com/giljant/red-app-mobile.git
cd red-app-mobile
npm install
npx expo start
```

Skeniraj QR kod s Expo Go appom.
